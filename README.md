# Подсчёт гласных букв в строке

Программа на C#, которая запрашивает у пользователя текст и подсчитывает количество гласных букв (русских) в нём.

## Описание

Пользователь вводит произвольную строку. Программа:
1. Проверяет, не пустая ли строка (с помощью `IsNullOrEmpty`).
2. Перебирает все символы строки.
3. Для каждого символа проверяет, является ли он гласной буквой (используя `Contains`).
4. Выводит общее количество гласных.

## Использование

1. Запустите программу.
2. Введите текст (можно на русском языке).
3. Программа выведет количество гласных букв.

### Пример

```
Введите текст: Привет, как дела?

Гласных букв: 6
```

## Объяснение ключевых методов

### `string.IsNullOrEmpty(text)`

Это статический метод класса `string`. Он возвращает `true`, если строка равна `null` или является пустой (`""`).  
**Зачем нужен?**  
В программе мы обращаемся к символам строки через `text[i]`. Если строка пустая, такой доступ вызовет исключение. `IsNullOrEmpty` позволяет безопасно проверить ввод и вывести понятное сообщение об ошибке, не допуская аварийного завершения.

### `string.Contains(char)`

Метод `Contains` проверяет, содержится ли указанный символ (или подстрока) в данной строке. Возвращает `true`, если символ найден, иначе `false`.  
**Зачем нужен?**  
В программе есть строка `glasne_all`, содержащая все гласные буквы (и строчные, и заглавные). Для каждого символа из введённого текста мы проверяем: `glasne_all.Contains(letter)`. Если `true` — значит, символ является гласной, и счётчик увеличивается.

## Детали реализации

- Строка гласных: `"аеёиоуыэюяАЕЁИОУЫЭЯ"` — включает как строчные, так и заглавные буквы, поэтому регистр не влияет на подсчёт.
- Цикл `for` проходит по всем индексам строки от `0` до `text.Length - 1`.
- Переменная `count` накапливает результат.
- После подсчёта результат выводится на экран.

## Примечания

- Если ввести пустую строку, программа сообщит об ошибке и завершится без подсчёта.
- Программа учитывает только русские гласные буквы. Для английских нужно изменить строку `glasne_all`.
- Можно улучшить программу, добавив поддержку обеих раскладок или используя готовые методы классификации символов (например, `Char.IsLetter`), но текущий вариант наглядно демонстрирует работу `Contains`.
